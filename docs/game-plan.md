# KiiiKiii — Game Plan v1.0

> 버전: 1.0 (Phase 0 → 1 기준)
> 작성일: 2026-04-29
> 기반: 기획위원회 1차·2차 합의사항 + 전략보고서
> 플랫폼: LINE Mini App (LIFF) / TypeScript + Phaser 3

---

## 1. 서비스 정체성

### 제품 정의
**"K-pop 팬덤 소셜 컴패니언 서비스"**

싸이월드 미니홈피 구조 × KiiiKiii 3등신 캐릭터 × 일본 오시카츠 문화
게임(머지 퍼즐)은 목적이 아닌 수단 — 아이템 획득 엔진 역할

### 코어 태그라인
> "KiiiKiii와 가까워지고 싶은 모든 행동이 응모권이 되고,
> 응모권으로 KiiiKiii와 더 가까워진다"

### 타겟
- 주: 일본 여성 팬, 20~30대
- 보조: 오시카츠 문화 경험자 (콘서트·굿즈·이벤트 참여 습관 보유)
- 플랫폼 도달: LINE 9,500만 MAU × 기존 KiiiKiii 팬덤

---

## 2. 코어 루프 (2단 구조)

```
[1단] 머지 퍼즐 플레이
  → 게임 클리어 → 의상/소품 아이템 획득 + XP + 응모권

[2단] 방(공간) 꾸미기
  → 아이템 배치 → 소셜 방문 유도
  → 다른 팬 파도타기 → 방명록/포토부스
  → 외부 동기(남들이 보니까) → 다시 게임

[연결] 친밀도 시스템
  일일 활동(게임·방문·포토부스·일정 확인) → XP → 레벨업 → 슬롯 확장
```

## 5. 모듈 상세

### 5-1. 홈 (내 방)
- 싸이월드 미니홈피 스타일 방 구조
  - 배경 (벽지/테마)
  - 바닥
  - 가구 레이어 (배치 가능 아이템)
  - 소품 레이어
- 내 캐릭터(최애 3등신)가 방 안에 존재
- 친밀도 바 + 레벨 표시
- 빠른 액션: Today 바로가기 / 게임 바로가기
- 방문자 수 표시 (소셜 압력)

### 5-2. 오늘 (Today)
- KiiiKiii 공식 일정 캘린더 자동 D-Day 카운트
- 뉴스 피드 (컴백 소식, 팬미팅 공지 등)
- 운영툴 1회 입력 → 자동 D-Day 계산
- 포토카드 없음 → 3등신 일러스트 카드 대체
- 응모권 획득 CTA 자연 노출

**Today 일정 예시 데이터 (Phase 0 하드코딩)**
```
D-7  콘서트 "WAIKIIIKIII Tour" — 도쿄 국제포럼
D-14 팬미팅 응모 마감
D-21 새 앨범 WAIKIIIKIII 발매
```

### 5-3. 게임 (머지 퍼즐)
- 스테이지 방식 머지 퍼즐 (2048-like merge mechanic)
- 하트 소모형: 1스테이지 = 하트 1개 소모
- 보상 구조
  - 클리어 → 의상/소품 아이템 + XP + 응모권
  - 퍼펙트 클리어(보너스 조건) → 희귀 아이템 확률 UP
- 스테이지 진행: 1~10 (Phase 0 데모용)
- 레벨 10마다 한정 衣装 해금

**Phase 0 밸런스 설정**
```
시작 하트: 3개
하트 자연 회복: 30분당 1개
게임 XP: 스테이지 클리어 +10, 퍼펙트 +5 추가
코인: 스테이지 클리어 +20~50
응모권: 스테이지 클리어 +1, 5연속 클리어 +1 보너스
```

### 5-4. 탐색 (파도타기 + 소셜)

**파도타기**
- 같은 최애 팬 우선 노출
- 무제한 무료 (하트 미소모)
- 다른 팬 방을 내 캐릭터를 데리고 방문
- 방문 시 방명록 남기기 가능

**소셜 메카닉 3종**

| 기능 | 내용 | 과금 요소 |
|------|------|---------|
| 파도타기 | 다른 팬 방 방문, 최애 기준 매칭 | 무료 |
| 방명록(선물 남기기) | 방문 시 게임 아이템 소모 → 선물 | 아이템 소모 |
| 포토부스 | 포즈→배경→SNS 공유, 10초 내 완료 | 프리미엄 프레임 유료 |

**포토부스 3단계**
1. 포즈 선택 (기본 5종 무료, 프리미엄 포즈 유료)
2. 배경 선택 (기본 무료, 시즌 한정 배경 유료)
3. SNS 공유 (Instagram/X/LINE 기본 무료)

**프라이버시**
- 기본: 같은 최애 팬에게 공개
- 설정: 전체 공개 / 팔로워만 / 비공개 선택 가능

## 6. 응모권 BM 시스템

### 핵심 철학
> 팬의 모든 행동에 응모권이 붙는다. 응모권은 KiiiKiii와의 특별한 경험으로 교환된다.

### 응모권 획득 경로
| 행동 | 응모권 획득 |
|------|------------|
| 머지 스테이지 클리어 | +1 |
| 5연속 클리어 보너스 | +1 |
| 파도타기 방문 (일일 5회) | +1/회 |
| Today 일정 확인 | +1/일 |
| 포토부스 공유 | +2 |
| QR 현장 체크인 | +5 |
| 팬덤 패스 구독자 일일 보너스 | +3/일 |

### 응모권 사용 (티어 구조)
| 티어 | 필요 응모권 | 상품 예시 |
|------|-----------|---------|
| 브론즈 | 10장 | 디지털 굿즈 (배경화면, 스티커) |
| 실버 | 30장 | 실물 굿즈 (포토카드, 아크릴 키홀더) |
| 골드 | 100장 | 팬미팅 우선 응모권, 굿즈 세트 |
| 플래티넘 | 300장 | 콘서트 특별석, 음방 방청 |
| 다이아 | 1,000장 | 화상 통화, 현장 팬 미팅 |

*응모권 이월: 당월 미사용 → 50% 다음 달 이월 (소멸 압력 유지)*

### 법적 구조 (설계 전제)
- 경품표시법 적합: 응모=추첨 구조, 과금 연동 없는 무료 획득 경로 확보
- 일본 소비자청 가이드 기준 설계 (법무 검토 전제)

## 12. 스타쉽 설득 핵심 논거

1. **IP 자산 창출**: 3등신 캐릭터 = TinyTAN처럼 KiiiKiii 독자 IP. 공동 소유 구조 제안 가능
2. **운영 부담 최소**: 일정 1회 입력 → 자동 D-Day. 포토카드 검수 없음 (일러스트 카드 대체)
3. **즉각 도달**: LINE 9,500만 MAU × 기존 팬덤 = Day 1 도달 가능
4. **확장성**: 타 아티스트 온보딩 플랫폼으로 발전 가능 (스타쉽 전체 아티스트)
5. **일본 타이밍**: 오시카츠 문화 전성기 × LINE 지배력 × K-pop 붐 교차점
